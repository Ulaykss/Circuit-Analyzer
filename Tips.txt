Запуск программы осуществляется через терминал
Сначала нужно активировать виртуальное окружение командной:
.\venv\Scripts\Activate.ps1
Деактивация виртуального окружения:
deactivate

Если не использовать виртуальное окружение, то нужно установить недостающие библиотеки, для этого нужна команда:
pip install -r requirements.txt

Для запуска программы следует прописать команду:
python main.py

Для сборки Python проекта в .exe:
pyinstaller --onefile --noconsole main.py


Примечание: некоторые файлы создались автоматически при сборки проекта в exe файл
